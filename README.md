## 📌 Project Title
Clothing Size Recommendation Using Machine Learning

## 🎯 Motivation (Problem & Dataset)

Online shoppers often struggle to choose the right clothing size due to differences in sizing standards across brands and the inability to physically try items before purchase. This leads to incorrect size selection, low customer satisfaction, and high product return rates for e-commerce platforms.

To address this problem, we use the **Clothing Fit Dataset for Size Recommendation** from Kaggle, which contains approximately 82,000 records of real e-commerce clothing purchases with 14 features. The dataset includes customer measurements (such as height, weight, and body type), product attributes (category, size, and brand), and a fit label indicating whether the item was **Fit**, **Small**, or **Large**.

By applying machine learning techniques to this dataset, our goal is to build an intelligent system that predicts the most suitable clothing size for users, helping customers make better size decisions and reducing return rates for online retailers.
