Dataset Name:
Clothing Fit Dataset for Size Recommendation

Kaggle URL:
https://www.kaggle.com/datasets/rmisra/clothing-fit-dataset-for-size-recommendation

License:
CC0: Public Domain (Free to use for research and educational purposes)

Original Purpose:
This dataset was created to analyze clothing size fit in e-commerce platforms.
It is used to build machine learning models that predict whether a clothing item
fits correctly (Fit), is too small (Small), or too large (Large) based on
customer measurements and product attributes.

File Format:
The original dataset is provided in JSON format and contains two JSON files.
For this project, the JSON files were converted into CSV format to facilitate
data exploration, preprocessing, and modeling.
